# 임시 객체

```cpp

class A
{
public:
    void print() {std::cout << "Hello <<}
}

int main()
{
    int n(1) // n이라는 이름의 변수에 1이라는 값 할당

    int(1)   // 임시 객체 -> 1이라는 값 할당된 이름없는 객체
    // 이름을 지어 놓지 않았기 때문에 잠깐 생겼다 사라짐.
    
    A a;     // a라는 이름의 객체
    a.print(); 

    A();     // 익명 객체 // A형식으로 생성된 이름없는 객체
    A().print(); // 이름 없는 A 타입의 익명객체가 생성된 후, 함수 실행 후 사라짐.

}
```
익명 객체는 R-value 로서 작동한다.
